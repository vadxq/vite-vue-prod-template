# Vue 3 + Typescript + Vite

Vue3 工程化项目实践demo

- 编程语言：TypeScript 4.2.4
- 构建工具：Vite 2
- 前端框架：Vue 3.0.11
- 路由工具：Vue Router 4.0.6
- 状态管理：Vuex 4.0.0
- CSS：Sass + Postcss
- HTTP 工具：Axios
- MOCK: mockjs + vite plugins
- Git Hook 工具：Husky + Lint-staged
- 代码规范：EditorConfig + Prettier + ESLint
- 提交规范：Commitlint
